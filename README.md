# CauMon: リアルタイムSTL監視ツール

Signal Temporal Logic (STL) を使ったリアルタイム監視ツールです。Web UIから簡単にサーバーを起動でき、TCPでデータを受信して、MATLABでロバストネスと因果関係を計算します。

## ✨ 主な機能

- 🌐 **Web UI**: ブラウザから監視サーバーを起動・停止
- 📡 **リアルタイム監視**: TCPでデータを受信し、即座にSTL評価
- 📊 **自動グラフ更新**: MATLABで可視化、リアルタイムで更新
- 🎯 **動的設定**: シグナル名とSTL式をWeb UIから変更可能
- 🔍 **因果関係分析**: 違反/満足の原因シグナルを特定
